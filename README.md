# discord-vanity-url-checker
basit bir url checker vanities kısmına URL ekleyerek hangisinin boş hangisinin dolu olduğunu görebilirsiniz ben bir kaç tane ekledim örnek olarak available.txt de bos olan urlleri gosteriyor unavailable.txt ellemenize gerek yok bilerek ekledim cunku silerseniz 429 hatasi veriyor yani ellemeyin install.bat'ı açarak modulleri yukleyebilirsiniz run.bat'ı acarak kullanabilirsiniz.
**not:** Eğer rate limit yerseniz, yani 429 hatası alırsanız, VPN kullanarak IP’nizi değiştirin.
